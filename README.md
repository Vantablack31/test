# STMP Tools

Local browser-based inspector for `.stmp` seatmap files.

## Features

- Load `.stmp` / XML files locally
- Count physical seats
- Render seats from `gx` / `gy`
- Inspect row, seat, level, section, and page
- Filter by row and seat number
- Export summary as JSON

## Usage

1. Save `index.html`
2. Open it in a browser
3. Load a `.stmp` file
4. Inspect seats visually

## Notes

This version is read-only. It does not write changes back into the XML yet.
